Генератор номеров из диапозона целых чисел (from some testcase)
===

1. Сделать функцию которая выводит каждый новый вызов случайное число от 1 до 20 и на каждой попытке уменьшает
количество выбираемых значений на 1.

Тоесть для диапазона чисел [0...4]:

```
[1,2,3,4] => (rand result) => 2
[1,3,4] => (rand result] => 1
[3,4] => (rand result) => 3
[4] => (num 1 ->then w/o rand ->return 4) => 4
```

2. Сделать функцию которая не повтряет прошлые последовательности значений.
То есть если в одной сессии генерируем чтото из этих строк:

{2,1,4,3}

То в следующей либо конкурентной нельзя выдавать повторы для таких случаев:

{2,1,4,3} -> заменить другой последовательностью

Для этого установим константу QUERY_CHAIN_LENGTH = 20 (т.е. 20 вопросов может быть максимально задано)
